# VRC-Releases

VRC-Releases — репозиторий для отслеживания и публикации релизов VRChat-проекта.

## Описание

Этот репозиторий предназначен для хранения и распространения релизов. Здесь публикуются актуальные версии проекта с описанием изменений.

## Установка

1. Перейдите в раздел [Releases](../../releases).
2. Скачайте нужную версию.
3. Следуйте инструкциям из раздела установки конкретного релиза.

## Использование

После установки следуйте документации, прилагаемой к каждому релизу.
